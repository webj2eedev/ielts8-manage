
1、与旧工会会计制度比较，新工会会计制度下新增的资产类科目是（   ）。
A 投资 B 固定资产 C 其他应收款 D 待处理财产损溢
解析：D 《工会会计制度》待处理财产损溢是指工会在资产清查过程中查明的各种资产盘盈、盘亏和报废、毁损的价值。反之，各种资产的盘盈、盘亏、报废、毁损均需要经过本科目进行核算。

2、与旧工会会计制度比较，新工会会计制度下取消的资产类科目是（   ）。
A 应收下级经费 B 其他应收款 C 投资 D 借出款 
解析：D 《工会会计制度》新制度没有设置“借出款”科目，但设置了“其他应收款”科目。转账时，工会应当将原账的“借出款”科目余额，转入新账的“其他应收款”科目。

3、与旧工会会计制度比较，新工会会计制度下增加的负债类科目是（   ）。
A 长期待摊费用 B 代管经费 C 应付职工薪酬 D 应付工资（离退休费）
解析： C《工会会计制度》新制度设置了“应付职工薪酬”科目，该科目的核算内容包含了原账“应付工资（离退休费）”、“应付地方（部门）津贴补贴”、“应付其他个人收入”科目的核算内容。转账时，工会应当将原账的“应付工资（离退休费）”、“应付地方（部门）津贴补贴”、“应付其他个人收入”科目余额转入新账的“应付职工薪酬”科目。

4、与旧工会会计制度比较，新工会会计制度中科目名称不变但是对原核算内容进行修订的会计科目有（   ）。
A 其他应收款 B 业务支出 C 其他应付款 D 借入款 E 借出款
解析：ABC 《工会会计制度》其他应收款和其他应付款均并入了借出款或者借入款，业务支出包括培训费、会议费、专项业务费和其他业务支出等，具体核算内容细化和调整。

5、根据《工会新旧会计制度有关衔接问题的处理规定》，某县级以上工会在新旧衔接中，经分析认定，结余资金中包含已经作为项目管理的需要结转下年继续使用的项目资金 2000万元。对于其中符合工会资金结转性质的1500万元转入新账时，借方科目是（   ）。
A 工会资金结转 B 工会资金结余 C 结余 D 财政资金结余 
解析：C 新制度未设置“结余”科目，但设置了“工会资金结转”“工会资金结余”“财政拨款结转”“财政拨款结余”科目。
6、根据《工会新旧会计制度有关衔接问题的处理规定》，新旧衔接工作中，县级以上工会一级科目和基层工会基础科目分别为（   ）。
A 45和25 B 43和23 C 45和24 D 42和24
解析：C 《工会新旧会计制度有关衔接问题的处理规定》 科目的设置要完整，县级以上工会组织的科目为 45 个一级科目，基层工会组织基础科目 24 个。

7、根据《工会新旧会计制度有关衔接问题的处理规定》，以下科目中，新旧衔接时将原账会计科目余额转入新账，根据原账科目分析填列的是（   ）。
A 其他应收款 B 库存现金 C 投资 D 应收上级经费 
解析：D 《工会新旧会计制度有关衔接问题的处理规定》 转账时，工会应当按照相关规定将原账的“应收上级经费”科目余额中的工会建会筹备金转入新账的“其他应收款”科目，将原账的“应收上级经费”科目余额减去建会筹备金后的差额转入新账的“应收上级经费”科目。

8、根据《工会新旧会计制度有关衔接问题的处理规定》，以下科目中，新旧衔接时将原账会计科目余额转入新账，根据原账科目分析填列的是（   ）。
A 财政应返还额度 B 银行存款 C 应付下级经费 D 其他应付款 
解析：C 《工会新旧会计制度有关衔接问题的处理规定》转账时，工会应当按照相关规定将原账的“应付下级经费”科目余额中的工会建会筹备金转入新账的“其他应付款”科目，将原账的“应付下级经费”科目余额减去建会筹备金后的差额转入新账的“应付下级经费”科目。

9、根据《工会新旧会计制度有关衔接问题的处理规定》，新旧衔接中，将下列原账会计科目余额转入新账时，科目金额可能发生变化的是（   ）。
A 银行存款 B 零余额账户用款额度 C 固定资产 D 投资
解析：C 《工会新旧会计制度有关衔接问题的处理规定》 固定资产转为库存物品或无形资产

10、根据《工会新旧会计制度有关衔接问题的处理规定》，根据新会计制度，无形资产需要在（   ）科目下设置明细科目进行核算。
A 固定资产 B 累计折旧 C 资产基金 D 专用基金
解析：C 《工会新旧会计制度有关衔接问题的处理规定》借记无形资产，贷记资产基金——无形资产。

11、根据《工会新旧会计制度有关衔接问题的处理规定》，在新旧工会会计制度衔接前的准备阶段，按计划在2021年处理完毕。某工会资产清查发现现金盘亏800元，报批后进行会计处理涉及的科目是（   ）。
A 借记待处理财产损溢 B 贷记待处理财产损溢 C 借记其他支出 D 贷记其他收入
解析：C 《工会新旧会计制度有关衔接问题的处理规定》借记其他支出 贷记库存现金

12、根据《工会会计制度》，以下项目中，未明确列入工会代管经费的是（   ）。
A 党费 B 乒乓球社团会费 C 摄影兴趣小组活动经费 D 职工缴纳的互助医疗金
解析：A《工会新旧会计制度有关衔接问题的处理规定》新制度设置了“库存现金”、“银行存款”科目，其核算内容不再包括工会代管的应当单独建账核算的党费、团费等。

13、根据《工会会计制度》，以下项目中，关于工会代管经费衔接工作论述错误的是（   ）。
A 新旧衔接中对于不符合制度规定的代管经费，应当对相应余额予以转销   B 新旧衔接中，将原账会计科目余额转入新账阶段，存在代管经费的，应当将原账“库存现金”“银行存款”科目余额中属于代管经费的金额，转入新账“库存现金”“银行存款”科目下的“代管经费”明细科目   C 工会组织缴纳的职工互助保险记入代管经费   D 新旧衔接工作转账阶段，工会应当将原账的“代管经费”科目余额，转入新账的“代管经费”科目
解析：C《工会新旧会计制度有关衔接问题的处理规定》代管经费是指其他组织委托工会代管的有指定用途的、不属于工会收入的资金，如代管的社团活动费、职工互助保险等。代管经费要严格按照资金的用途使用，并严格履行规定的报批程序。属于收入性质业务范畴的，如工会收到的各种救灾、救助捐赠款项，需统一调配使用的，不应在代管经费中核算，要计入工会收入，纳入工会预算收支统一管理。

14、根据《工会新旧会计制度有关衔接问题的处理规定》，在新旧衔接中，基层工会应该分析原账“结余”科目余额，将符合结转性质的工会资金余额转入新账（   ）科目。
A 工会资金结转 B 工会资金结余 C 财政资金结转 D 财政资金结余
解析：A《工会新旧会计制度有关衔接问题的处理规定》新制度未设置“结余”科目，但设置了“工会资金结转”“工会资金结余”“财政拨款结转”“财政拨款结余”科目。
15、根据《工会新旧会计制度有关衔接问题的处理规定》，在新旧衔接中，县级以上工会应该分析原账“结余”科目余额，将符合结转性质的财政资金余额转入新账（   ）科目。
A 工会资金结转 B 工会资金结余 C 财政资金结转 D 财政资金结余
解析：C《工会新旧会计制度有关衔接问题的处理规定》。新制度未设置“结余”科目，但设置了“工会资金结转”“工会资金结余”“财政拨款结转”“财政拨款结余”科目。
16、根据《工会新旧会计制度有关衔接问题的处理规定》，在新旧衔接中，县级以上工会应该分析原账“结余”科目余额，将符合工会资金结转性质的资金余额转入新账（   ）科目。
A 工会资金结转 B 工会资金结余 C 财政资金结转 D 财政资金结余
解析：A《工会新旧会计制度有关衔接问题的处理规定》新制度未设置“结余”科目，但设置了“工会资金结转”“工会资金结余”“财政拨款结转”“财政拨款结余”科目。
17、根据《工会新旧会计制度有关衔接问题的处理规定》，在新旧衔接中，县级以上工会将基建账套相关数据并入新账过程中，将基建账科目余额转入新账并进行分析调整，以上工作最后出现借方差额时，应计入新账（   ）科目。
A 工会资金结转 B 工会资金结余 C 财政资金结转 D 财政资金结余
解析：B《工会新旧会计制度有关衔接问题的处理规定》。新制度执行后，基本建设工程项目均通过新账的“在建工程”科目核算，工会不再执行《国有建设单位会计制度》（财会字〔1995〕45 号）。新旧制度转换时，工会应当将基建账相关数据并入新账。完成资产、负债和净资产处理后，按照上述新账科目的借方合计金额减去贷方合计金额后的差额，贷记或借记新账中“工会资金结余”科目。


18、根据《工会新旧会计制度有关衔接问题的处理规定》，在新旧衔接中，某基层工会将原账的“库存物品”科目余额直接转入新账的“库存物品”科目时，应按照原账“库存物品”科目余额，将（   ）科目对应余额转入新账的“资产基金—库存物品”科目。
A 工会资金结转 B 工会资金结余 C 结余 D 财政资金结余 
解析：C《工会新旧会计制度有关衔接问题的处理规定》新制度设置了“库存物品”科目，并规定库存物品取得时应当同时确认资产基金。转账时，工会应当将原账的“库存物品”科目余额直接转入新账的“库存物品”科目，同时，按照原账“库存物品”科目余额，将“结余”科目对应余额转入新账的“资产基金—库存物品”科目。
19、根据《工会新旧会计制度有关衔接问题的处理规定》，工会在新旧制度转换时，应当将 2021 年 12 月 31 日前未入账的负债按照新制度规定记入新账。登记新账时，按照确定的负债金额，借记（   ）科目，贷记相关负债科目。
A 工会资金结转 B 工会资金结余 C 结余 D 财政资金结余 
解析：B 《工会新旧会计制度有关衔接问题的处理规定》 工会在新旧制度转换时，应当将 2021 年 12 月 31 日前未入账的负债按照新制度规定记入新账。登记新账时，按照确定的负债金额，借记“工会资金结余”科目，贷记相关负债科目。

20、根据《工会新旧会计制度有关衔接问题的处理规定》，某县级以上工会在新旧衔接中，经分析认定，结余资金中包含已经作为项目管理的需要结转下年继续使用的项目资金 2000万元。对于其中符合财政拨款结转性质的 500万元转入新账时，借方科目是（   ）。
A 工会资金结转 B 工会资金结余 C 结余 D 财政资金结转 
解析：C 《工会新旧会计制度有关衔接问题的处理规定》新制度未设置“结余”科目，但设置了“工会资金结转”“工会资金结余”“财政拨款结转”“财政拨款结余”科目。县级以上工会应当根据分析结果，将符合财政拨款结转性质的资金余额转入“财政拨款结转”科目；将符合财政拨款结余性质的资金余额转入“财政拨款结余”科目；将符合工会资金结转性质的资金余额转入“工会资金结转”科目；将剩余的资金余额转入“工会资金结余”科目。
